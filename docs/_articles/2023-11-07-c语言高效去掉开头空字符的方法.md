### 前言

看到很多使用迭代起吊空字符的，觉得代发写的太费劲。这里提供一段使用strspn实现的代码

### 简单介绍strspn

strspn位于string.h,姊妹函数strcspn和strtok  
函数声明：size_t strspn(const char *str1, const char *str2)；  
此函数用于从str1总找到不包含在str2中的任何字符，并返回index。

具体的介绍可自行搜索，也可参考[c plus plus函数介绍](http://www.cplusplus.com/reference/cstring/strspn/)。  
废话不多说，直接上代码。这里假定空白字符包含：空格，\r,\n,\r,\v,\f这6个字符。
[code] 
    #include <string.h>
    #include <stdint.h>
    
    char* trim(char* objStr){
    	uint32_t idx;
    	idx = strspn(objStr, " \r\n\t\v\f");
    	return (objStr+idx);
    }
    
    /*这里可以测试一下*/
    int main(void){
    	char test[] = "\r\n\t    It is a long long ago...\r\n";
    	printf("original:\r\n%s", test);
    	printf("trimed:\r\n%s", trim(test));
    }
    
    
[/code]

测试结果如下：
[code] 
    original:
    
                It is a long long ago...
    trimed:
    It is a long long ago...
    
[/code]
