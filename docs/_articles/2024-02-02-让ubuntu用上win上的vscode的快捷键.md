# 让ubuntu用上win上的vscode的快捷键
不知道什么时候（反正以前没感觉）ubuntu上的vscode使用的快捷键可在windows上区别很大。尤其是代码查看前进和后退的时候。在windows上是`alt` + `<-/->`，在ubuntu上是`ctrl` + `alt` + `+/-`.
搜索之后发现安装Windows default keybindings插件就可以了。
